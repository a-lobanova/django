# Road Excel Processor

Веб-приложение на Django для загрузки Excel файлов с данными об автомобильных дорогах
и генерации отчета в формате Word.

## Стек

- Python
- Django
- Pandas
- python-docx

## Запуск

git clone https://github.com/a-lobanova/django

cd django

pip install -r requirements.txt

python manage.py runserver

## Запуск Тестов 

python manage.py test
